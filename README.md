# 🏫 School Management Admin Dashboard — Built with Next.js & Tailwind

A modern and responsive **school management dashboard** built with **React**, **Next.js 15**, and **Tailwind CSS**.  
It features beautiful layouts and dynamic components tailored for four types of users: **Admin**, **Teacher**, **Student**, and **Parent**.

> 🔗 **Live Demo**: [school-management-app-steel.vercel.app/admin](https://school-management-app-steel.vercel.app/admin)

## ✨ Features

- ⚙️ Role-based dashboards: Admin, Teacher, Student, and Parent
- 📱 Fully responsive layout using Tailwind CSS
- 📊 Interactive and animated charts with **Recharts**
- 📅 Integrated calendar and scheduling
- 📋 Dynamic list tables with pagination
- 🧾 Forms built with **React Hook Form** and **Zod** for validation
- 📂 Clean and reusable components & structure
- ⚡ Optimized with **Next.js 15 App Router** and lazy loading

---

## 🛠️ Tech Stack

| Technology       | Description                           |
|------------------|---------------------------------------|
| **Next.js 15**   | React framework with App Router       |
| **Tailwind CSS** | Utility-first CSS framework           |
| **Recharts**     | Chart library for React               |
| **React Hook Form** | Modern form management             |
| **Zod**          | Type-safe validation schema           |
| **React Big Calendar** | Calendar and scheduling UI     |

## 🚀 Getting Started

```bash
# Clone the project
git clone https://github.com/your-username/school-dashboard.git

# Navigate into the project directory
cd school-dashboard

# Install dependencies
npm install

# Run the development server
npm run dev
````

Then open [http://localhost:3000](http://localhost:3000) in your browser.

---

## 🧭 Pages Structure

* `/admin` — Admin Dashboard
* `/teacher` — Teacher Dashboard
* `/student` — Student Dashboard
* `/parent` — Parent Dashboard

Each dashboard contains relevant features, charts, and data tables designed for that user role.
